实现细节：

1.生成训练数据，我生成了0～N内数字两两组合，然后做异或。两两组和的数字和异或的结果都转换成了二进制，并用M个int来存储，其中trainX为N*Nx2*M维，trainY为N*NxM维。

2.构建神经网络 输入层:2*M,隐藏层：H,输出层:M

3.训练网络

4.测试结果
